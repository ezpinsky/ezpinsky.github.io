WordSpot

WordSpot is a fast-paced word-finding game similar to boggle with a powerful depth-first search algorithm used to generate new boards based on user input. Find as many words as you can by connecting adjacent letters in a 4 x 4 letter grid.

BurnReads

BurnReads is a GoodReads inspired place on the internet to belittle, banish, and torch books that make your eyes burn! Never again will you have to worry about organizing all the books that you hate. Simply search for a book to set its torch status, add it to a custom kindling shelf, or leave a fiery bad review!

InstaClone

Instaclone is a social media application modeled on Instagram built using React, Flask, PostgreSQL and SQLAlchemy. It currently supports image uploading, photo feed, follows, likes, and comments.
